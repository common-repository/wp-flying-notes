== Description ==

Put draggable notes in your wordpress blog.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug in, you will see an new option called "WP Flying Notes".

** Adding message **
5. Go to the "WP Fliying Slider" option in the Settings menu.
6. Click in the "Add New Flying Note"
7. Goto to the main page of your blog.
8. You will se the yellow notes flying around.
